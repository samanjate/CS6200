 * Running the program
    To run the program, please copy paste the .py file and the G1.txt file
    in the working directory of your python IDLE and use commands as follows:

    Task 2: execfile('pageRank.py')

    The programs might take a few seconds to run.
    The outputs will be recorded in files namely G1Perplexity.txt and 
    G1PRScore.txt in the same directory.

    To run the program on wtg file, please copy the file in the working directory
    and change the file name in the program to the file name with the extension.
